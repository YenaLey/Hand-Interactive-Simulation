프로젝트 개요
- 이 프로젝트는 Three.js를 사용하여 인터랙티브한 손 모델을 구현하였습니다. 사용자 인터페이스를 통해 손의 각 부분을 실시간으로 조절할 수 있습니다.

구현 기능
- 전체 손 인터페이스 구현: 손 전체 모델이 Three.js 씬에 렌더링되었습니다.
- 손마디 조절: 각 손가락의 관절을 슬라이더를 통해 개별적으로 조절할 수 있습니다.
- 손목 제어: 손목을 꺾었다 폈다가 할 수 있으며, 구부러진 손을 기준으로 손바닥을 회전시킬 수 있습니다.
- 엄지 회전: 엄지가 안쪽으로 자연스럽게 회전하도록 설정되었습니다.
- 손가락 동작: 손가락을 양쪽으로 펴고 오므릴 수 있는 기능이 추가되었습니다.
- 손 색상 설정: GUI를 통해 손의 색상을 변경할 수 있습니다.

사용 방법
- 슬라이더 조작: 각 슬라이더를 움직여 손의 특정 관절이나 손목, 손가락의 움직임을 조절하세요.
- 색상 변경: GUI에서 색상 선택기를 사용하여 손의 색상을 원하는 대로 변경할 수 있습니다.
- 카메라 제어: 마우스를 사용하여 손 모델을 회전시키거나 확대/축소할 수 있습니다.
